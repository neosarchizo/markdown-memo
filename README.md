# Markdown Memo App

Mobile memo application built with TypeScript, Expo and React Native Paper. Stores memos in Markdown format with rich editing capabilities, local storage, and comprehensive export features.

## Setup

```bash
npm install
```

## Run

```bash
npm start
# or
npx expo start
```

Press `a` for Android, `i` for iOS, or scan QR code with Expo Go app.

## Project Structure

- `app/` - Expo Router screens
- `src/` - Source code
  - `types/` - TypeScript interfaces
  - `contexts/` - React Context providers
  - `services/` - Business logic
  - `hooks/` - Custom React hooks
  - `components/` - Reusable UI components
  - `utils/` - Helper functions
- `documents/` - Project documentation (PRD, LLD, PLAN)
